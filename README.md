# CodeAlpha_Hangman_Game

A simple and interactive terminal-based Hangman game built using Python.

## Description

Hangman is a word guessing game where the player tries to guess a hidden word one letter at a time. The player has only 6 incorrect guesses before the game ends. This project is designed with a clean terminal interface, multiple categories, score tracking, replay option, and input validation.

---

## Features

- Multiple word categories
- Random word selection
- 6 maximum wrong guesses
- Word progress display using underscores
- High score system
- Replay option
- Input validation
- Screen clearing after each turn
- Simple colored terminal output
- Clean and minimal UI

---

## Categories

- Animals
- Fruits
- Countries
- Technology

Each category contains exactly 5 predefined words.

---

## Technologies Used

- Python
- Built-in modules only:
  - `random`
  - `os`
  - `time`

---
